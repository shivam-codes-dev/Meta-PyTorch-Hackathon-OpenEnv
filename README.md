# Meta OpenEnv RL Submission
This is a Reinforcement Learning environment built for the Meta x Scaler Hackathon.

## Environment Details
- **Goal:** The agent must reach the target (4,4) from the starting position (0,0).
- **Library:** Built using PyTorch and NumPy.
- **Reward System:** +10 for reaching the goal, -0.1 for each step to encourage efficiency.

## How to Run
The logic is contained in `submission.py`.
